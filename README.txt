The Monkey Market Simulator is a stock market simulation environment where autonomous trading agents, styled as monkeys, trade various commodities using distinct behavioral strategies. The project models how individual trading decisions collectively influence stock prices over multiple rounds using a pseudo-supply and demand mechanic.

It features five traditional heuristic strategies (Random, Buy Low Sell High, Threshold, Hold, and Scared) alongside an advanced Neural Network agent that utilizes a minimax lookahead tree with alpha-beta pruning to predict market states and optimize its trading actions.

to run simply run python GUI.py
